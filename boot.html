Introduction

In DCP , user has to specify the parameter on which the password needs to keep on changing such as system time , user Geolocation , user age (up to days) and much more.

Working

Let us consider the system time as parameter for user authentication with “HH:MM” format,based on the password policies length of the password be 8 character,among these 8 characters let 4 characters be defined by user which is static in nature and the remaining 4 characters be defined based on the parameters chosen by the user ,here it is system time which is dynamic in nature
